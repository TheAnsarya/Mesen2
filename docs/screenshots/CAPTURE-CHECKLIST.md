# Screenshot Capture Checklist

## Purpose

Use this checklist to capture and track every screenshot anchor referenced by the user guides.

## Status Key

- `pending`: screenshot not captured yet
- `captured`: screenshot captured and placed at target path
- `verified`: screenshot captured and confirmed to match expected panel/state
